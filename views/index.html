Image Search Abstraction Layer

Instructions:

  • Use https://imgsrch-al.glitch.me/search/+ term you want to search +?offset=+ a number, which indicates page number.
  • Use https://imgsrch-al.glitch.me/latest-seach to view the past 8 searches made.

Example Input:

    https://imgsrch-al.glitch.me/search/lolcats%20funny?offset=1
